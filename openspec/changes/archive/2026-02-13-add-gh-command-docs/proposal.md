## Why

项目的某些技能（如 GitHub PR 相关技能）需要使用 `gh` (GitHub CLI) 命令，但当前的开发环境文档中缺少 `gh` 的安装和配置说明，导致开发者可能无法正确配置所需环境。

## What Changes

- 在 `docs/development-environment.md` 中添加 `gh` (GitHub CLI) 的安装说明
- 说明 `gh` 命令的用途（用于 GitHub PR 相关操作）
- 添加基本的安装命令和验证方法

### New Capabilities
*无*

### Modified Capabilities
*无*（仅文档内容补充，不涉及功能规格变化）

## Impact

- **文档**: `docs/development-environment.md` - 新增 `gh` 命令安装说明
- **开发环境**: 开发者需要安装 `gh` 才能使用项目的 GitHub 相关技能
